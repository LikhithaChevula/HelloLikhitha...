# Hello Likhitha

A tiny Python script that greets Likhitha and lists some nice qualities.

Usage

Run to list all compliments:

```bash
python3 hello_likhitha.py
```

Run to show one random compliment:

```bash
python3 hello_likhitha.py --random
```

You can modify `GOOD_THINGS` in `hello_likhitha.py` to add or change items.
